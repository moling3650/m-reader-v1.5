# m-reader-v2
小说阅读器 web app

## 项目流程

1. 用vue-cli构建项目的骨架
2. 抓取mock数据，用express编写api接口
3. 在客户调用api接口获取数据，有必要时使用jsonp和base64解密数据
4. 确定页面组件分层，编写路由和组件切换效果

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build
```
